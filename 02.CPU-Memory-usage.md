## CPU and Memory usage: 


### Show Top Processes by CPU usage:

_Quick recap of useful `top` options:_
- Press `P` : sort by CPU.
- Press `M` : sort by memory.
- Press `1` : view per-core CPU usage.
- Press `Shift + E` : change memory units (KB/MB/GB).
- Press `q` : quit.



_The `top` with processes sorted by CPU usage (highest first):_
```
top -o %CPU 
```


_Top 5 by CPU:_
```
ps -eo pid,comm,%cpu --sort=-%cpu | head -n 6
```


_Top 10 by CPU:_
```
ps -eo pid,ppid,cmd,%mem,%cpu --sort=-%cpu | head -n 11
```



### Show Top 10 Processes by Memory usage: 


_The `top` command with processes sorted by memory usage (descending order)._
- `-o` : tells top to order by a field.
- `%MEM` : the field is percentage of memory usage.
- Inside top, you’ll now see the highest RAM consumers at the top.


```
top -o %MEM
```


```
ps aux --sort -%mem
```


_Top 5 by Memory:_
```
ps -eo pid,comm,%mem --sort=-%mem | head -n 6
```


_Top 10 by Memory:_
```
ps -eo pid,ppid,cmd,%mem,%cpu --sort=-%mem | head -n 11
```
